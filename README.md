# FPL Dashboard MVP 🚀

A sleek, responsive, and blazing-fast Fantasy Premier League Dashboard built with **Next.js 15**, **React**, and **TypeScript**. 
Engineered with a stunning dark-mode Glassmorphism UI, this application seamlessly integrates with the official public FPL API endpoints to deliver real-time football analytics without requiring any sensitive passwords.

* **Live Rank Projections:** Intelligent heuristic model that calculates intra-gameweek rank movements, switching to "Official" status once FPL data is checked.
* **Transfers Analyzer:** Detailed breakdown of transfer hits, points impact, and chip usage.
* **Fixture Ticker:** Interactive visual heat-map of upcoming opponents mapped directly to your squad.
